# ⚡ One-click Setup Guide

Use this guide right after creating your vault from the template.

---

## ✅ First-run checklist note

- [ ] Create your vault from GitHub **Use this template**
- [ ] Open the vault folder in Obsidian
- [ ] Turn on **Community Plugins**
- [ ] Confirm **Restricted Mode** is off for this trusted vault
- [ ] Run `npm run plugins:install` (or install plugins manually: Dataview, Tasks, Templater, Calendar, Kanban)
- [ ] Enable all installed plugins
- [ ] In Templater settings, set **Template folder location** to `Templates`
- [ ] Open `00 Dashboard/Home Dashboard.md` and pin it as your start page
- [ ] Create your first project from `Templates/Project Template.md`
- [ ] Add one task in that project and tag top priority items with `🔥`

---

## 🧭 Onboarding flow

### 1) Bootstrap (5-10 min)
- Create vault from template
- Run `npm run plugins:install` and enable required plugins
- Configure Templater template path

### 2) Launch your workspace (5 min)
- Open `00 Dashboard/Home Dashboard.md`
- Open `00 Dashboard/Tasks Dashboard.md`
- Open `00 Dashboard/Projects Overview.md`
- Keep these as your default working tabs

### 3) Seed your system (10-15 min)
- Create 1-3 active projects in `01 Projects`
- Add tasks with due dates where possible
- Mark your top 1-3 tasks with `🔥` in the task text

### 4) Daily operating loop (5 min/day)
- Capture incoming work in your daily note
- Move meaningful work into project notes
- Execute from the task and home dashboards

### 5) Weekly reset (15-20 min/week)
- Run `00 Dashboard/Weekly Review.md`
- Close done tasks and archive stale work
- Re-prioritize active projects for the next week
